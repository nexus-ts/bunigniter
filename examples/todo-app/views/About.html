Features

  • SSR React Views — Pages are React components rendered on the server
  • SQLite Database — Lightweight, zero-config persistence
  • CRUD Operations — Create, read, update, delete todos
  • Filter & Search — By status (active/completed), priority, or text
  • Post/Redirect/Get — HTML form submissions with PRG pattern
